# 11. AI Insights & Analysis UI

**Status:** ✅ Done

- **AI Insights & Analysis Panel:** Integrated as a tab adjacent to the interactive transcript on the video detail page.
- **Tabbed Layout Switcher:** A clean segmented selector allowing users to switch between "AI Insights" (showing summary, key takeaways, and technologies) and "Interactive Transcript".
- **Status-Aware State Handling:**
  - \`pending\` / \`processing\` → spinner + pulsing skeleton cards reading "AI is analyzing the video transcript to generate insights…"
  - \`failed\` → red error card showing the saved generation error
  - \`completed\` → structured presentation showing:
    - **Summary**: Paragraph detailing the core context.
    - **Key Takeaways**: Bullet list of items with clean checkmark icons.
    - **Topics & Technologies**: Colored badges representing keywords extracted.
- **API Mapping & Polling:** Updated frontend client \`api.ts\` and \`types.ts\` to map \`aiSummary\` properties from the video details response. Expanded polling logic in \`VideoDetail.tsx\` to query every 3s as long as \`aiSummary.status\` is still processing, enabling live page updates without manually refreshing.

### Key Files

- \`src/app/videos/[id]/VideoDetail.tsx\` — tabbed panel switcher, AI loaders, key takeaway lists
- \`src/lib/types.ts\` — \`IAISummary\` mapping, \`PlayResult.aiSummary\` type
- \`src/lib/api.ts\` — updated \`getPlay\` to resolve \`aiSummary\`
