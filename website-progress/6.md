# 6. HLS Video Player (Video.js)

**Status:** ✅ Done

- **Component:** `HlsPlayer` — wraps Video.js in a React component
- **Player features:**
  - HLS playback (`application/x-mpegURL`)
  - Responsive + fluid layout
  - Playback speed selector (0.5x, 1x, 1.5x, 2x)
  - Skip forward/backward buttons (10s)
  - **Quality selector** — `videojs-contrib-quality-levels` + `videojs-hls-quality-selector` plugins for manual rendition switching (displays current quality)
- **Implementation details:**
  - Creates `<video-js>` element dynamically (avoids React hydration mismatch)
  - Plugins loaded via `require()` at module scope with `typeof window` guard (client-only)
  - Separate `useEffect` for player init (once) and source updates (on `src` change)
  - Proper cleanup: disposes player + removes DOM element on unmount
  - Custom `ExtendedPlayer` TypeScript interface for plugin methods
- Dark border + rounded corners + shadow with hover effect

- `src/components/HlsPlayer.tsx`
- `src/types/videojs.d.ts` — empty declaration to avoid module conflicts
