# 4. Upload Page

**Status:** ✅ Done

- **Route:** `/upload`
- **3-step upload flow:**
  1. `initiateUpload(title)` — reserves video record, gets presigned URL
  2. `uploadToStorage(url, file, onProgress)` — direct PUT to MinIO with progress bar
  3. `completeUpload(videoId)` — confirms to backend, kicks off pipeline
- **Phase state machine:** `idle → uploading → finalizing → done → error`
- **UI features:**
  - Title input (optional, defaults to filename)
  - Drag-and-drop style file picker (accepts `video/*`)
  - Selected file preview card (name, size, MIME type) with remove button
  - Progress bar during upload with percentage
  - "Finalizing…" state after upload completes
  - Error banner on failure
  - On success → auto-redirect to `/videos/:videoId`
- Inputs disabled during active upload
