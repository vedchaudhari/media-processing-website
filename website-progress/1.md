# 1. Project Setup & Configuration

**Status:** ✅ Done

- **Framework:** Next.js 16 (App Router) + React 19 + TypeScript
- **Styling:** Tailwind CSS v4 (PostCSS plugin)
- **Fonts:** Geist Sans + Geist Mono (Google Fonts via `next/font`)
- **Data fetching:** TanStack React Query v5
- **Video player:** Video.js v8
- **Env:** `NEXT_PUBLIC_API_BASE_URL` set in `.env.local` (defaults to `http://localhost:3000`)

### Key Files

- `next.config.ts` — default Next config (no custom settings yet)
- `src/app/globals.css` — Tailwind import + CSS custom properties for light/dark
- `src/app/layout.tsx` — root layout with nav bar (Library + Upload links)
- `src/lib/providers.tsx` — React Query provider (staleTime 2s, retry 1)
- `package.json`
