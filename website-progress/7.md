# 7. Reusable UI Components

**Status:** ✅ Done

### StatusBadge (`src/components/StatusBadge.tsx`)

- Pill-shaped badge showing video status (capitalized)
- Color-coded per status:
  - Gray → uploading/uploaded
  - Blue → inspecting/inspected/planning/planned
  - Amber → transcoding
  - Green → completed
  - Red → failed
- Shows transcoding progress percentage when applicable
- Dark mode aware

### VideoCard (`src/components/VideoCard.tsx`)

- Card component for the library grid
- **Thumbnail Layout:** Features a 16:9 aspect ratio image container at the top of the card.
- **Dynamic Thumbnail:** Displays the lazy-loaded video thumbnail when generated.
- **Fallback Placeholder:** Shows a modern gradient placeholder with a centered video icon (pulsing micro-animation while processing) when the thumbnail is not yet available.
- **Floating Badge:** Overlays the `StatusBadge` in the top-right corner of the thumbnail container.
- Shows title, status text, and transcoding progress bar below the thumbnail.
- All cards link to `/videos/:id` with interactive hover zoom and shadow transitions.
