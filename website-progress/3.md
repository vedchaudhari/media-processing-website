# 3. Video Library Page (Home)

**Status:** ✅ Done

- **Route:** `/` (home page)
- Fetches all videos via `useQuery` → `listVideos()`
- **Smart polling:** Auto-refetches every 3s while any video has an in-progress status; stops polling once all are settled (completed/failed)
- **States handled:**
  - Loading → 3 skeleton cards with `animate-pulse`
  - Error → red banner with error message + hint to check if backend is running
  - Empty → dashed border "No videos yet" with link to upload
  - Data → responsive grid of `VideoCard` components (1/2/3 columns)
- "Upload" button in header and page body
