# 5. Video Detail / Player Page

**Status:** ✅ Done

- **Route:** `/videos/[id]`
- **Server component** (`page.tsx`) awaits async params (Next 16 pattern) → passes `id` to **client component** (`VideoDetail.tsx`)
- Fetches playback info via `useQuery` → `getPlay(videoId)`
- **Smart polling:** Refetches every 3s while video is in-progress and not ready; stops when completed/failed
- **3 UI states:**
  - **Ready** → renders `HlsPlayer` with playback URL
  - **Failed** → red error card with failure message
  - **Processing** → spinner + current status (capitalized) + progress bar during transcoding phase + "This page updates automatically" hint
- Title + `StatusBadge` shown in header for all states
- "← Back to library" link at top

### Key Files

- `src/app/videos/[id]/page.tsx` — server component (param unwrap)
- `src/app/videos/[id]/VideoDetail.tsx` — client component (data fetch + render)
